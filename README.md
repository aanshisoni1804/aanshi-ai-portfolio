# Aanshi Soni — AI Portfolio

A responsive one-page portfolio showcasing my software engineering experience, AI projects, technical skills, and achievements.

## Live website

**[View the live portfolio](https://aanshi-soni-portfolio.argb01.chatgpt.site)**

## Highlights

- Custom AI-illustrated portrait used as the hero centrepiece
- Responsive layout for desktop and mobile
- Experience timeline covering Amazon, IIFL, and DRDO
- Featured AI/ML and computer vision projects
- Working contact, LinkedIn, GitHub, and Gmail links
- Accessible navigation and reduced-motion support

## Built with

- React / TypeScript
- Vinext (Next.js-compatible runtime)
- CSS and responsive media queries
- OpenAI image generation for the custom portrait
- ChatGPT Sites for deployment

## AI-assisted workflow

AI was used to generate the custom illustrated portrait, accelerate layout and copy exploration, write and refine the implementation, and test the responsive production build. All content was reviewed and adapted to match my own experience.

## Run locally

```bash
npm install
npm run dev
```

Then open the local URL shown in the terminal.
